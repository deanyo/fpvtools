# fpvtools

Simple homepage for FPV web tools.

Planned role:

- root landing page for `fpvtools.co.uk`
- links out to individual tools such as `rates.fpvtools.co.uk`
- keeps product-level copy separate from the individual tool repos

Current linked projects:

- `rates`: https://rates.fpvtools.co.uk
- `rates` source: https://github.com/deanyo/rates
- `blackbox-audio` source: https://github.com/deanyo/blackbox-audio

## Run Locally

```bash
python3 -m http.server 8000
```

Open:

```text
http://localhost:8000
```
